Совсем незаметно (почти) вышел Windows Server 2016.


Начать знакомиться с новой операционной системой можно скачав её с TechNet Evaluation Center (https://www.microsoft.com/en-us/evalcenter/evaluate-windows-server-2016?i=1) в виде ознакомительной версии на 180 дней.

На TechNet можно найти раздел "Windows Server 2016" (https://technet.microsoft.com/en-us/windows-server-docs/get-started/windows-server-2016-technical-preview-5) в котором содержатся важные сведения. Особенно рекомендую внимательно прочесть главы "
System Requirements". Возможно сэкономите себе много времени и не столкнётесь с проблемами как.



Так же, полагаю, будет интересно прочитать бесплатную книгу по Windows Server 2016, которая доступна по следующей ссылке: https://blogs.msdn.microsoft.com/microsoft_press/2016/09/26/free-ebook-introducing-windows-server-2016/
